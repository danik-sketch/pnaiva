# Ежедневник / To-Do List

Простое консольное приложение для управления списком задач. Помогает не забывать о важных делах в течение дня.

## Возможности

- **Добавление задач:** Создавай новые дела с описанием.
- **Просмотр:** Выводит список всех текущих задач с отметкой о статусе (сделано/не сделано).
- **Отметка о выполнении:** Помечай задачи как выполненные.
- **Редактирование:** Изменяй описание существующей задачи.
- **Удаление:** Убирай задачи из списка.
- **Сохранение:** Задачи автоматически сохраняются в файл и загружаются
